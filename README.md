# Trabalho Final - Programação Web Front-End

## Nome do Projeto
Cadastro de Jogos com Tema Dinâmico

## Nome da Dupla
- Gabriel Lopes
- Francisco Flávio

## Descrição
A página permite montar uma lista de jogos com nome, preço e prioridade, ajudando uma pessoa que queira se planejar para comprar seus jogos.  
Além disso, é possível alterar o visual da aplicação através de diferentes temas dinâmicos (Padrão, Dark, Neon e RDR2), aplicados diretamente via DOM.
